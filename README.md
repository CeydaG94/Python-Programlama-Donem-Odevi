# Python-Programlama-Donem-Odevi
PYTHON PROGRAMLAMA DÖNEM ÖDEVİ Konu: Akıllı Müşteri Yönetim ve Analiz Sistemi (Telco Senaryosu) Ortam: Google Colab

Akıllı Müşteri Yönetim ve Analiz Sistemi (Telco Senaryosu)
Bu proje, Python Programlama dersi dönem ödevi kapsamında hazırlanmıştır. 
Bir telekomünikasyon (Telco) şirketindeki müşteri verilerinin yönetilmesi, 
analiz edilmesi ve faturalandırma süreçlerinin simüle edilmesini amaçlar.

📁 Proje İçeriği
# Müşteri Veri Yapıları: Değişkenler, Listeler ve Sözlükler (Dictionary) kullanılarak veri yönetimi.

# VIP Karar Mekanizması: Koşullu ifadeler (If-Else) ile müşteri segmentasyonu.

# Fonksiyonel Hesaplamalar: KDV dahil fatura tutarı hesaplayan özel fonksiyonlar.

# Veri Analizi ve Temizleme: set() yapısı ile benzersiz hizmetlerin tespiti.

# Churn Analizi: Müşteri kayıp riski tespiti için döngüsel kontroller.

🛠 Kullanılan Teknolojiler ve Kütüphaneler
Python 3.x

Google Colab (Geliştirme ortamı)

math: Matematiksel yuvarlama işlemleri için.

datetime: Güncel tarih bilgisi eklemek için.

random: Benzersiz Müşteri ID (CustomerID) üretimi için.

🚀 Nasıl Çalıştırılır?
Repoda bulunan .ipynb uzantılı dosyayı indirin.

Google Colab üzerine yükleyin.

Hücreleri sırasıyla (Shift + Enter) çalıştırın.

📝 Kritik Sorular ve Yanıtlar
Neden Sözlük (Dictionary) tercih edildi? Verilere anahtar-değer (key-value) ilişkisiyle daha hızlı ve anlamlı erişmek için.

Churn Riski nasıl hesaplandı? Aktif olmayan müşteriler veya sadakat süresi 6 aydan az olan kullanıcılar for döngüsü ile taranarak "Riskli" olarak işaretlendi.
